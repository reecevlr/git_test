* Git Cheatsheet
---

*** Cloning
---
In a repository:
  1. Click on <>Code Button
  2. Copy SSH Key

In terminal:
  1. Switch to appropriate directory
  2. git clone <SSH KEY>

*** Workflow
---
  1. git add
       - git add .
         - period represents every change in current directory
       - git add <file_name.txt> 
  2. git commit -m "important changes made"
  3. git status
  4. git log
